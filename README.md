# Little Book Heaven System

## Project Description

The **Little Book Heaven System** is a Java-based console application designed to manage book inventory efficiently.
It allows users to perform core operations such as adding, updating, searching, and deleting book records.

This project demonstrates the use of **Object-Oriented Programming (OOP)** principles and file handling in Java to build a structured and functional system.


## Technologies Used

Java
Object-Oriented Programming (OOP)
File Handling (Text Files)
NetBeans / Eclipse IDE


## Features

Add new books to the system
Search books by different criteria
Update existing book details
Delete books from inventory
Store and retrieve data using text files


## How to Run

1. Open the project in NetBeans or Eclipse
2. Navigate to the main class:

   ```
   thelittlebookhaven.java
   ```
3. Run the program
4. Use the console menu to interact with the system

## Purpose

This project was developed to:

* Apply core OOP concepts such as classes and objects
* Understand file handling in Java
* Practice building a structured application
* Simulate a real-world inventory management system
